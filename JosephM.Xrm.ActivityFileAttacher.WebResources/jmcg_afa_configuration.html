Joseph McGregor Activity File Attacher

This solution adds a web resource to simplify attaching files to email or appointment records

The web resource allows drag and drop, or selection of, one or more files to attach to the activity without having to go through the several step ootb attachment dialog

To configure the web resource for use add the web resource named jmcg_afa_activityfileattacher.html to either the email or appointment forms. These settings seem to provide best results

  • Display label on the form = False
  • Restrict cross-frame scripting, where supported = False
  • Scrolling = Never
  • Display border = False
